# Todo Application

## Описание
Это простая реализация Todo приложения для практики в Kotlin и TypeScript

## Запуск
### Backend
```shell
cd todo-api
./gradlew generateJooqClasses
./gradlew nativeCompile
./build/native/nativeCompile/todo-api
```

## Технологии
### Backend
- Kotlin
- Spring Book
- JOOQ
- Flyway
- PostgreSQL
- MapStruct
